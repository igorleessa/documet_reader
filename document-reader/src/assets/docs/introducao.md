# Introdução ao Document Reader

Bem-vindo ao **Document Reader** — seu visualizador de documentos Markdown!

## O que é este projeto?

Este projeto foi criado para facilitar a visualização de documentações escritas em formato `.md` (Markdown), transformando textos simples em páginas bem formatadas e fáceis de ler.

## Principais Funcionalidades

- 📂 **Listagem automática** de documentos disponíveis
- 🎨 **Renderização visual** com formatação rica
- 🔗 **Navegação fácil** entre documentos
- 💻 **Código com destaque de sintaxe**

## Como Usar

1. Adicione seus arquivos `.md` na pasta `src/assets/docs/`
2. Registre o arquivo no serviço `DocService`
3. Acesse a aplicação e clique no documento desejado
